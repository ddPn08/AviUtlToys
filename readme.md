Aviutil Toys は Aviutil をより便利にするユーティリティのセットです。

# Features

- 🗃️ Aviutil ファイル管理機能 (プラグインやスクリプト等)
- 📁 Exo ファイルの管理機能。
- 🤐 Softalk を使った音声合成機能
- 🔌 プラグインシステム
