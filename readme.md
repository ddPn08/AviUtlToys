<h1 align="center">Aviutil Toys</h1>

Aviutil Toys は Aviutil をより便利にするユーティリティのセットです。

> **Note**  
> 現在開発中です。  
> 近日中にベータ版をリリースする予定です。  

## Features  
- 🗃️Aviutilファイル管理機能 (プラグインやスクリプト等)
- 📁Exoファイルの管理機能。
- 🔌プラグインシステム
